configs/config

port

Default port number for running Server. Default value is 3000 which is picked if there is no PORT env variable set.

port

splitterAddress

P2PSP splitter bind address. By default '127.0.0.1' is picked if nothing is supplied. Set via env - SBINDADDRESS.

splitterAddress

splitterBin

P2PSP splitter binary path. Set via env - SPLITTERBIN.

splitterBin

monitorBin

P2PSP monitor binary path. Set via env - MONITORBIN.

monitorBin
